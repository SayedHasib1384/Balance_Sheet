# Balance Sheet Project

A simple Balance Sheet application built with HTML, CSS, and JavaScript, using LocalStorage to save data locally in the browser.  
This project allows users to track financial assets, liabilities, and equity persistently without a server.

## Features
- Add, edit, and delete assets, liabilities, and equity entries
- Automatic calculation of totals
- Responsive design using CSS
- Persistent data using browser LocalStorage

## Project Structure
- `index.html` : main HTML file
- `style.css` : project styles
- `script.js` : JavaScript logic including LocalStorage functionality
